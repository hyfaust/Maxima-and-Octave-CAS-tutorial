第5章：泛函分析应用 ​

★★★★☆ 高级难度

概述 ​

泛函分析是数学分析的一个重要分支，它研究的是函数的函数（泛函）的极值问题。本章将介绍变分法的基础知识、欧拉-拉格朗日方程、泛函极值问题及其在力学中的应用。

先决条件

学习本章前，建议先掌握以下内容：

  • 微积分基础（特别是积分和微分）
  • 常微分方程基础
  • 基本的线性代数知识
  • 对物理学（特别是力学）有基本了解

1. 变分法基础 ​

1.1 什么是变分法？ ​

ℹ️ 变分法变分法是研究泛函极值问题的数学方法。与普通函数求极值不同，变分法处理的是依赖于整个函数（而不仅仅是点）的量。

变分法的核心思想是：寻找一个函数 y(x)y(x)y(x)，使得某个依赖于 y(x)y(x)y(x) 及其导数的泛函 J[y]J[y]J[y] 达到极值。

泛函是从函数空间到实数的一个映射。例如：

J[y]=∫abF(x,y,y′) dxJ[y] = \int_a^b F(x, y, y') \, dxJ[y]=∫ab​F(x,y,y′)dx

其中 FFF 是已知函数，y=y(x)y = y(x)y=y(x) 是未知函数，y′=dydxy' = \frac{dy}{dx}y′=dxdy​。

1.2 变分的基本概念 ​

给定一个函数 y(x)y(x)y(x)，考虑其小扰动 y(x)+ϵη(x)y(x) + \epsilon \eta(x)y(x)+ϵη(x)，其中 ϵ\epsilonϵ 是一个小参数，η(x)\eta(x)η(x) 是满足边界条件 η(a)=η(b)=0\eta(a) = \eta(b) = 0η(a)=η(b)=0 的任意函数。

泛函 J[y]J[y]J[y] 的变分定义为：

δJ=ddϵJ[y+ϵη]∣ϵ=0\delta J = \frac{d}{d\epsilon} J[y + \epsilon \eta] \Big|_{\epsilon=0}δJ=dϵd​J[y+ϵη]​ϵ=0​

如果 δJ=0\delta J = 0δJ=0，则称 y(x)y(x)y(x) 使得泛函 JJJ 取得极值。

1.3 泛函极值的必要条件 ​

设 y(x)y(x)y(x) 在 [a,b][a, b][a,b] 上连续可微，且 y(a)=yay(a) = y_ay(a)=ya​，y(b)=yby(b) = y_by(b)=yb​ 固定。考虑泛函：

J[y]=∫abF(x,y,y′) dxJ[y] = \int_a^b F(x, y, y') \, dxJ[y]=∫ab​F(x,y,y′)dx

则 y(x)y(x)y(x) 使 JJJ 取得极值的必要条件是 y(x)y(x)y(x) 满足欧拉-拉格朗日方程：

∂F∂y−ddx(∂F∂y′)=0\frac{\partial F}{\partial y} - \frac{d}{dx} \left( \frac{\partial F}{\partial y'} \right) = 0∂y∂F​−dxd​(∂y′∂F​)=0

2.2 特殊形式的欧拉-拉格朗日方程 ​

情况1：FFF 不显含 xxx

如果 F=F(y,y′)F = F(y, y')F=F(y,y′)，则欧拉-拉格朗日方程可以简化为：

F−y′∂F∂y′=CF - y' \frac{\partial F}{\partial y'} = CF−y′∂y′∂F​=C

其中 CCC 是常数。这称为贝尔特拉米恒等式。

情况2：FFF 不显含 yyy

如果 F=F(x,y′)F = F(x, y')F=F(x,y′)，则欧拉-拉格朗日方程简化为：

∂F∂y′=C\frac{\partial F}{\partial y'} = C∂y′∂F​=C

情况3：FFF 不显含 y′y'y′

如果 F=F(x,y)F = F(x, y)F=F(x,y)，则欧拉-拉格朗日方程简化为：

∂F∂y=0\frac{\partial F}{\partial y} = 0∂y∂F​=0

2.3 自然边界条件 ​

当边界值 y(a)y(a)y(a) 或 y(b)y(b)y(b) 未指定时，需要添加自然边界条件：

3. 泛函极值问题 ​

3.1 含参数的泛函极值 ​

考虑含参数 λ\lambdaλ 的泛函：

J[y]=∫abF(x,y,y′,λ) dxJ[y] = \int_a^b F(x, y, y', \lambda) \, dxJ[y]=∫ab​F(x,y,y′,λ)dx

极值条件为：

∂F∂y−ddx(∂F∂y′)=0且∂J∂λ=0\frac{\partial F}{\partial y} - \frac{d}{dx} \left( \frac{\partial F}{\partial y'} \right) = 0 \quad \text{且} \quad \frac{\partial J}{\partial \lambda} = 0∂y∂F​−dxd​(∂y′∂F​)=0且∂λ∂J​=0

3.2 条件极值问题（等周问题） ​

等周问题：在满足约束条件

∫abG(x,y,y′) dx=L\int_a^b G(x, y, y') \, dx = L∫ab​G(x,y,y′)dx=L

的函数中，寻找使泛函

J[y]=∫abF(x,y,y′) dxJ[y] = \int_a^b F(x, y, y') \, dxJ[y]=∫ab​F(x,y,y′)dx

取极值的函数 y(x)y(x)y(x)。

解法：引入拉格朗日乘子 λ\lambdaλ，构造新泛函：

J~[y]=∫ab(F+λG)dx\tilde{J}[y] = \int_a^b \left( F + \lambda G \right) dxJ~[y]=∫ab​(F+λG)dx

然后求解欧拉-拉格朗日方程：

∂(F+λG)∂y−ddx(∂(F+λG)∂y′)=0\frac{\partial (F + \lambda G)}{\partial y} - \frac{d}{dx} \left( \frac{\partial (F + \lambda G)}{\partial y'} \right) = 0∂y∂(F+λG)​−dxd​(∂y′∂(F+λG)​)=0

3.3 泛函极值的充分条件 ​

二阶变分：

δ2J=d2dϵ2J[y+ϵη]∣ϵ=0\delta^2 J = \frac{d^2}{d\epsilon^2} J[y + \epsilon \eta] \Big|_{\epsilon=0}δ2J=dϵ2d2​J[y+ϵη]​ϵ=0​

对于泛函极值：

  • 如果 δ2J>0\delta^2 J > 0δ2J>0，则 JJJ 取得局部极小值
  • 如果 δ2J<0\delta^2 J < 0δ2J<0，则 JJJ 取得局部极大值

勒让德条件：

设 Fy′y′′=∂2F∂y′2F_{y'y''} = \frac{\partial^2 F}{\partial y'^2}Fy′y′′​=∂y′2∂2F​，则：

  • Fy′y′′≥0F_{y'y''} \geq 0Fy′y′′​≥0 是极小值的必要条件
  • Fy′y′′≤0F_{y'y''} \leq 0Fy′y′′​≤0 是极大值的必要条件

4. 力学中的应用 ​

4.1 拉格朗日力学 ​

在经典力学中，系统的运动由拉格朗日函数 LLL 描述：

L=T−VL = T - VL=T−V

其中 TTT 是动能，VVV 是势能。

哈密顿原理（最小作用量原理）：系统的实际运动路径使作用量泛函

S[q]=∫t1t2L(q,q˙,t) dtS[q] = \int_{t_1}^{t_2} L(q, \dot{q}, t) \, dtS[q]=∫t1​t2​​L(q,q˙​,t)dt

取极值，其中 qqq 是广义坐标，q˙=dqdt\dot{q} = \frac{dq}{dt}q˙​=dtdq​ 是广义速度。

应用欧拉-拉格朗日方程得到运动方程：

∂L∂q−ddt(∂L∂q˙)=0\frac{\partial L}{\partial q} - \frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}} \right) = 0∂q∂L​−dtd​(∂q˙​∂L​)=0

4.2 哈密顿力学 ​

定义广义动量：

p=∂L∂q˙p = \frac{\partial L}{\partial \dot{q}}p=∂q˙​∂L​

哈密顿量定义为：

H(q,p,t)=pq˙−LH(q, p, t) = p \dot{q} - LH(q,p,t)=pq˙​−L

哈密顿正则方程：

q˙=∂H∂p,p˙=−∂H∂q\dot{q} = \frac{\partial H}{\partial p}, \quad \dot{p} = -\frac{\partial H}{\partial q}q˙​=∂p∂H​,p˙​=−∂q∂H​

4.3 应用实例 ​

例1：最速降线问题

在重力场中，寻找从点 AAA 到点 BBB 的曲线，使得质点沿该曲线滑下的时间最短。

设曲线为 y(x)y(x)y(x)，从 (0,0)(0,0)(0,0) 到 (1,h)(1,h)(1,h)。质点的速度由能量守恒给出：

v=2gyv = \sqrt{2gy}v=2gy​

滑下时间泛函为：

T[y]=∫011+y′22gy dxT[y] = \int_0^1 \frac{\sqrt{1 + y'^2}}{\sqrt{2gy}} \, dxT[y]=∫01​2gy​1+y′2​​dx

应用欧拉-拉格朗日方程可解得摆线方程。

例2：悬链线问题

在重力场中，寻找两端固定、长度固定的柔软链条在平衡状态下的形状。

设链条密度为 ρ\rhoρ，长度为 LLL，两端点为 (x1,y1)(x_1, y_1)(x1​,y1​) 和 (x2,y2)(x_2, y_2)(x2​,y2​)。

势能泛函为：

V[y]=ρg∫x1x2y1+y′2 dxV[y] = \rho g \int_{x_1}^{x_2} y \sqrt{1 + y'^2} \, dxV[y]=ρg∫x1​x2​​y1+y′2​dx

约束条件为长度固定：

∫x1x21+y′2 dx=L\int_{x_1}^{x_2} \sqrt{1 + y'^2} \, dx = L∫x1​x2​​1+y′2​dx=L

这是一个条件极值问题，应用拉格朗日乘子法可解得悬链线方程。

Octave 示例 ​

matlab
% 变分法基础示例
% 计算泛函 J[y] = ∫(y^2 + y'^2)dx 的极值函数

% 定义符号变量
syms x y(x) C1 C2;

% 定义被积函数 F(x,y,y')
F = y(x)^2 + diff(y(x))^2;

% 计算欧拉-拉格朗日方程
F_y = diff(F, y(x));
F_yp = diff(F, diff(y(x)));
EL_eq = F_y - diff(F_yp, x) == 0;

% 求解微分方程
y_sol = dsolve(EL_eq, y(0) == 0, y(1) == 1);

% 显示结果
disp('极值函数为：');
disp(y_sol);
