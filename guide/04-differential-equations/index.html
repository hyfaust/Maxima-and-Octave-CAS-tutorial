第4章：微分方程 ​

★★★☆☆

概述 ​

微分方程是描述自然现象和工程问题的核心数学工具。本章将介绍如何使用计算机代数系统（CAS）求解各类微分方程，包括常微分方程（ODE）和偏微分方程（PDE）的基础概念。

什么是微分方程？ ​

微分方程是包含未知函数及其导数的方程。根据导数类型的不同，可分为：

**常微分方程（ODE）**
：仅包含一个自变量的导数
**偏微分方程（PDE）**
：包含多个自变量的偏导数

基本概念 ​

  • 阶数：方程中出现的最高阶导数的阶数
  • 解：满足微分方程的函数
  • 通解：包含任意常数的解族
  • 特解：满足初始/边界条件的特定解

1. 一阶常微分方程 ​

1.1 可分离变量方程 ​

形如 dydx=f(x)g(y)\frac{dy}{dx} = f(x)g(y)dxdy​=f(x)g(y) 的方程称为可分离变量方程。

求解步骤：

  1. 分离变量：1g(y)dy=f(x)dx\frac{1}{g(y)} dy = f(x) dxg(y)1​dy=f(x)dx
  2. 两边积分：∫1g(y)dy=∫f(x)dx\int \frac{1}{g(y)} dy = \int f(x) dx∫g(y)1​dy=∫f(x)dx

示例：求解 dydx=xy\frac{dy}{dx} = xydxdy​=xy

分离变量：1ydy=xdx\frac{1}{y} dy = x dxy1​dy=xdx

积分：ln⁡∣y∣=x22+C\ln|y| = \frac{x^2}{2} + Cln∣y∣=2x2​+C

解：y=Cex2/2y = Ce^{x^2/2}y=Cex2/2

Octave
matlab
% 使用符号包求解可分离变量方程
pkg load symbolic

% 定义符号变量
syms x y(x)

% 定义微分方程：dy/dx = x*y
ode = diff(y, x) == x * y;

% 求解微分方程
y_sol = dsolve(ode);

% 显示通解
disp('通解:');
disp(y_sol);

% 求满足初始条件 y(0)=1 的特解
y_specific = dsolve(ode, y(0) == 1);
disp('满足 y(0)=1 的特解:');
disp(y_specific);

通解：y=1x2(∫x2⋅x2dx+C)=1x2(x55+C)=x35+Cx2y = \frac{1}{x^2} \left( \int x^2 \cdot x^2 dx + C \right) = \frac{1}{x^2} \left( \frac{x^5}{5} + C \right) = \frac{x^3}{5} + \frac{C}{x^2}y=x21​(∫x2⋅x2dx+C)=x21​(5x5​+C)=5x3​+x2C​

1.3 恰当微分方程 ​

形如 M(x,y)dx+N(x,y)dy=0M(x,y)dx + N(x,y)dy = 0M(x,y)dx+N(x,y)dy=0 的方程，若满足 ∂M∂y=∂N∂x\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}∂y∂M​=∂x∂N​，则称为恰当方程。

求解方法：存在函数 F(x,y)F(x,y)F(x,y) 使得 dF=Mdx+NdydF = Mdx + NdydF=Mdx+Ndy，则通解为 F(x,y)=CF(x,y) = CF(x,y)=C。

示例：求解 (2xy+3)dx+(x2−1)dy=0(2xy + 3)dx + (x^2 - 1)dy = 0(2xy+3)dx+(x2−1)dy=0

验证恰当性：∂∂y(2xy+3)=2x\frac{\partial}{\partial y}(2xy + 3) = 2x∂y∂​(2xy+3)=2x，∂∂x(x2−1)=2x\frac{\partial}{\partial x}(x^2 - 1) = 2x∂x∂​(x2−1)=2x，相等，所以是恰当方程。

求解：F=∫(2xy+3)dx=x2y+3x+g(y)F = \int (2xy + 3)dx = x^2y + 3x + g(y)F=∫(2xy+3)dx=x2y+3x+g(y)

∂F∂y=x2+g′(y)=x2−1\frac{\partial F}{\partial y} = x^2 + g'(y) = x^2 - 1∂y∂F​=x2+g′(y)=x2−1，所以 g′(y)=−1g'(y) = -1g′(y)=−1，g(y)=−yg(y) = -yg(y)=−y

通解：x2y+3x−y=Cx^2y + 3x - y = Cx2y+3x−y=C

2. 高阶常微分方程 ​

2.1 常系数线性齐次方程 ​

二阶常系数线性齐次方程：y′′+py′+qy=0y'' + py' + qy = 0y′′+py′+qy=0

特征方程：r2+pr+q=0r^2 + pr + q = 0r2+pr+q=0

根据特征根的情况：

  • 两个不同实根 r1,r2r_1, r_2r1​,r2​：y=C1er1x+C2er2xy = C_1 e^{r_1 x} + C_2 e^{r_2 x}y=C1​er1​x+C2​er2​x
  • 重根 rrr：y=(C1+C2x)erxy = (C_1 + C_2 x)e^{rx}y=(C1​+C2​x)erx
  • 共轭复根 α±βi\alpha \pm \beta iα±βi：y=eαx(C1cos⁡βx+C2sin⁡βx)y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)y=eαx(C1​cosβx+C2​sinβx)

示例：求解 y′′−3y′+2y=0y'' - 3y' + 2y = 0y′′−3y′+2y=0

特征方程：r2−3r+2=0r^2 - 3r + 2 = 0r2−3r+2=0，解得 r1=1r_1 = 1r1​=1，r2=2r_2 = 2r2​=2

通解：y=C1ex+C2e2xy = C_1 e^x + C_2 e^{2x}y=C1​ex+C2​e2x

求解方法：通解 = 齐次方程通解 + 非齐次方程特解

示例：求解 y′′−3y′+2y=e3xy'' - 3y' + 2y = e^{3x}y′′−3y′+2y=e3x

3. 偏微分方程基础 ​

3.1 偏微分方程的分类 ​

二阶线性偏微分方程的一般形式： A∂2u∂x2+2B∂2u∂x∂y+C∂2u∂y2+D∂u∂x+E∂u∂y+Fu=GA\frac{\partial^2 u}{\partial x^2} + 2B\frac{\partial^2 u}{\partial x \partial y} + C\frac{\partial^2 u}{\partial y^2} + D\frac{\partial u}{\partial x} + E\frac{\partial u}{\partial y} + Fu = GA∂x2∂2u​+2B∂x∂y∂2u​+C∂y2∂2u​+D∂x∂u​+E∂y∂u​+Fu=G

根据判别式 Δ=B2−AC\Delta = B^2 - ACΔ=B2−AC：

  • Δ>0\Delta > 0Δ>0：双曲型（波动方程）
  • Δ=0\Delta = 0Δ=0：抛物型（热传导方程）
  • Δ<0\Delta < 0Δ<0：椭圆型（拉普拉斯方程）

3.2 经典偏微分方程 ​

波动方程 ​

∂2u∂t2=c2∂2u∂x2\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}∂t2∂2u​=c2∂x2∂2u​

描述振动、波动现象。

热传导方程 ​

∂u∂t=k∂2u∂x2\frac{\partial u}{\partial t} = k \frac{\partial^2 u}{\partial x^2}∂t∂u​=k∂x2∂2u​

描述热量扩散、粒子扩散现象。

拉普拉斯方程 ​

∇2u=∂2u∂x2+∂2u∂y2=0\nabla^2 u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0∇2u=∂x2∂2u​+∂y2∂2u​=0

描述稳态温度分布、静电场等。

3.3 分离变量法简介 ​

分离变量法是求解偏微分方程的基本方法之一。

基本思想：假设解可以写成各个自变量函数的乘积。

示例：求解一维波动方程的初边值问题 ${∂2u∂t2=c2∂2u∂x2,0<x<L,t>0u(0,t)=u(L,t)=0,t>0u(x,0)=f(x),∂u∂t(x,0)=0,0<x<L \begin{cases} \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}, & 0 < x < L, t > 0 \\ u(0,t) = u(L,t) = 0, & t > 0 \\ u(x,0) = f(x), \frac{\partial u}{\partial t}(x,0) = 0, & 0 < x < L \end{cases} ⎩⎨⎧​∂t2∂2u​=c2∂x2∂2u​,u(0,t)=u(L,t)=0,u(x,0)=f(x),∂t∂u​(x,0)=0,​0<x<L,t>0t>00<x<L​$

设 u(x,t)=X(x)T(t)u(x,t) = X(x)T(t)u(x,t)=X(x)T(t)，代入方程分离变量得： $T′′c2T=X′′X=−λ \frac{T''}{c^2 T} = \frac{X''}{X} = -\lambda c2TT′′​=XX′′​=−λ$

求解两个常微分方程，利用边界条件得到特征值和特征函数。

4. 边值问题 ​

4.1 边值问题的概念 ​

边值问题是在微分方程基础上，给定边界条件（而非初始条件）的问题。

典型形式： ${y′′=f(x,y,y′),a<x<by(a)=α,y(b)=β \begin{cases} y'' = f(x, y, y'), & a < x < b \\ y(a) = \alpha, y(b) = \beta \end{cases} {y′′=f(x,y,y′),y(a)=α,y(b)=β​a<x<b$

4.2 打靶法（Shooting Method） ​

基本思想：将边值问题转化为初值问题，通过调整初始斜率使解满足另一端的边界条件。

算法步骤：

  1. 猜测初始斜率 y′(a)=s0y'(a) = s_0y′(a)=s0​
  2. 求解初值问题得到 y(b;s0)y(b; s_0)y(b;s0​)
  3. 根据 y(b;sk)−βy(b; s_k) - \betay(b;sk​)−β 调整 sk+1s_{k+1}sk+1​
  4. 重复直到满足精度要求

4.3 有限差分法（Finite Difference Method） ​

基本思想：用差商近似导数，将微分方程转化为代数方程组。

二阶中心差分： $y′′(xi)≈yi−1−2yi+yi+1h2 y''(x_i) \approx \frac{y_{i-1} - 2y_i + y_{i+1}}{h^2} y′′(xi​)≈h2yi−1​−2yi​+yi+1​​$

示例：求解边值问题 ${y′′=−4y+4x,0<x<1y(0)=0,y(1)=2 \begin{cases} y'' = -4y + 4x, & 0 < x < 1 \\ y(0) = 0, y(1) = 2 \end{cases} {y′′=−4y+4x,y(0)=0,y(1)=2​0<x<1$

将区间 [0,1][0,1][0,1] 分成 nnn 等份，步长 h=1/nh = 1/nh=1/n，得到线性方程组： $yi−1−2yi+yi+1h2=−4yi+4xi \frac{y_{i-1} - 2y_i + y_{i+1}}{h^2} = -4y_i + 4x_i h2yi−1​−2yi​+yi+1​​=−4yi​+4xi​$

整理为矩阵形式求解。

5. 数值解与符号解 ​

5.1 符号解的优势 ​

  • 给出精确的解析表达式
  • 便于理论分析和推导
  • 可以研究参数对解的影响

5.2 数值解的优势 ​

  • 适用于复杂方程（无解析解的情况）
  • 计算速度快
  • 可以处理实际工程问题

5.3 CAS中的微分方程求解 ​

Octave：

Maxima：

  • 数值解：使用 rk 函数（龙格-库塔法）

  4. 对于复杂PDE，考虑数值方法（有限差分、有限元等）

7.3 验证解的正确性 ​

  1. 将解代入原方程验证
  2. 检查边界/初始条件
  3. 使用不同方法求解比较结果
  4. 绘制解曲线观察合理性
